# Agent notes

Per-ticket output notes from agent runs (pattern borrowed from Filterful). One file per run:

```text
wow-XXX-<role>-<topic>.md
```

Examples: `wow-003-creative-tech-integrator-simulator-implementation.md`, `wow-004-frontend-ui-designer-ui-audit.md`, `wow-003-reviewer-review.md`.

Each note: date, ticket, agent, what was done/found, verdict (reviewers), demo/verification steps (implementers), open questions. Notes are append-only history — don't rewrite past notes; add new ones.
